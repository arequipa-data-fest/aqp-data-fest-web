# Actores del Sistema

Se definen los perfiles que interactúan con el sistema, centrando su rol en los procesos de negocio del evento.

| Código | Actor | Descripción |
| :--- | :--- | :--- |
| **ACT-01** | **Visitante** | Usuario no registrado que accede al sitio para informarse sobre la propuesta de valor, agenda y ponentes del evento. |
| **ACT-02** | **Participante** | Usuario que ha completado su registro mediante la plataforma externa (Luma) y regresa al sitio para obtener beneficios post-evento (certificados). |
| **ACT-03** | **Administrador** | Responsable técnico o de organización encargado de la gestión de metadatos, carga de activos multimedia y habilitación del sistema de certificados. |
