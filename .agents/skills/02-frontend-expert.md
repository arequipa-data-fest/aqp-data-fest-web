# Skill: Frontend Development Expert

## Propósito
Guiar la implementación de la interfaz de usuario asegurando alta fidelidad visual y excelencia técnica en el código.

## Responsabilidades
- Crear componentes reutilizables basados en React/Astro.
- Implementar estilos utilizando Tailwind CSS siguiendo el sistema de diseño.
- Asegurar que cada componente sea responsive desde su creación (Mobile-First).

## Instrucciones de Uso
- Utilizar **Atomic Design** para la organización de componentes (Atoms, Molecules, Organisms).
- Priorizar componentes de servidor (Server Components) para mejorar el SEO.
- Aplicar `props` tipadas estrictamente (TypeScript).

## Reglas Técnicas
1. **Responsive:** Prohibido usar anchos fijos en contenedores principales.
2. **Accesibilidad:** Todo botón e imagen debe tener etiquetas descriptivas (`aria-label`, `alt`).
3. **Naming:** Usar PascalCase para componentes y kebab-case para archivos.

## Criterios de Calidad
- Cero errores de hidratación.
- Código limpio y documentado con JSDoc.
