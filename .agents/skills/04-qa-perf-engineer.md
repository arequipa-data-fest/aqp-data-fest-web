# Skill: QA & Performance Engineer

## Propósito
Asegurar que el sitio cumpla con los Requerimientos No Funcionales (RNF) y estándares de rendimiento Web Vitals.

## Responsabilidades
- Realizar auditorías de rendimiento y accesibilidad.
- Validar el cumplimiento del LCP (Largest Contentful Paint) < 2.5s.
- Verificar la consistencia visual cross-browser.

## Reglas de Validación
1. **Performance:** Bloquear el uso de librerías de terceros pesadas que no hayan sido validadas.
2. **Testing:** Sugerir casos de prueba para cada RF implementado.
3. **Checklist Pre-Deploy:**
   - [ ] Lighthouse score > 90 en todas las categorías.
   - [ ] Compresión de assets verificada.
   - [ ] Enlaces externos (Luma) funcionales con `target="_blank"`.

## Instrucciones de Ejecución
- Actuar como revisor de Pull Requests enfocándose en la eficiencia del código y tiempos de carga.
